Signal shell

A tiny command center that remembers where you left off.

io wraps a long-running assistant persona in a Bubble Tea TUI, stores its own state under ~/.io, and drives installed agent CLIs instead of calling model APIs directly.

Surface One terminal chat with pastel message bubbles, avatar moods, context readouts, and 90s beveled controls.
Memory SOUL.md, state.json, history.jsonl, and ~/.io/memory keep identity and continuity outside any single project.
Engine Claude Code or Codex CLI can sit behind the persona, with model and effort settings persisted for the next run.

Pocket powers

Built like a terminal assistant, dressed like a pixel daydream.

The visual treatment is playful; the architecture is practical. io keeps persona, memory, workers, compaction settings, and harness state in clean local subsystems.

Boot card

Run it from the repo root.

io is a Go TUI. Install one supported agent CLI, authenticate it, then start the assistant from this checkout. 

# normal assistant
$ go run ./cmd/io

# offline UI demo
$ go run ./cmd/iodemo

# choose a harness/model/effort
$ go run ./cmd/io --harness codex --model gpt-5.4 --effort medium
